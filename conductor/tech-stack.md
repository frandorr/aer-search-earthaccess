# Tech Stack

## Language & Core
- **Python (>=3.13):** Modern Python with full support for typing and asynchronous features.
- **Polylith Architecture:** Modular monorepo structure.

## Libraries & Frameworks
- **Earthaccess (>=0.16.0, <0.17.0):** Primary integration for NASA Earthdata search and access.
- **aer-core:** The core library for the `aer` ecosystem.
- **Polylith CLI:** Workspace management and block creation.

## Tooling
- **uv:** Ultra-fast dependency and environment management.
- **Hatchling:** Build backend for Python projects.
- **Ruff:** All-in-one Python linter and formatter.
- **Basedpyright:** Modern, feature-rich type checker.
- **Pytest:** Workspace-wide testing framework.
- **Python Semantic Release:** Automated versioning and changelog generation.
